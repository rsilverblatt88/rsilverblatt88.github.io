Portfolio

Saga

Saga was our first group project, for which we built a bucket list web application. The thought behind the project was to ensure that we incorporate all of the skills which we learned at that point in the course. The application was built with Rails and incorporates HTML/CSS, JavaScript and PostgreSQL. The main thing I learned throughout the project was not only how to use each language, but also how to work together as a team. This was a very pleasurable experience as our team functioned very well together as tasks were divied up but at the same time team meetings were held daily to ensure that each team member was kept on the same page.

Hi5ive

Hi5ive was my final class project. The purpose of Hi5ive is to promote encouragement within the classroom by allowing classmates to send eachother "virtual" Hi5ives via SMS. Once a Hi5ive is received, it is displayed onto that classmates profile page which shows who sent it and what it was for. There is also a leadership board which displays the classmates with the most Hi5ives. The application was built with Rails and utilized Ruby, JavaScript, jQuery, Ajax, and PostgreSQL. I thoroughly enjoyed this project because it allowed me to guage how much I learned and was able to apply from a 3 month bootcamp experience.
